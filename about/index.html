PROJECT SHOWCASE

Build of Dawn: A Guide to MLBB Items and Hero Compatibility

A comprehensive academic project developed for the course WST101 - Web Systems and Technologies.

🎯 Project Objectives

✔

Provide detailed, high-quality descriptions of every in-game item and their strategic value.

✔

Establish clear hero-item recommendations to help players optimize their performance.

✔

Deliver an intuitive, responsive, and easy-to-use layout accessible across all devices.

✨ Key Features

Clear Item Explanations

Complex mechanics broken down into readable, actionable data.

Role-Based Suggestions

Tailored builds for Tanks, Mages, Assassins, and more.

Gameplay Optimization

Tools designed to improve decision-making during critical match phases.

👁 Scope & Limitations

PROJECT SCOPE

This project covers the current meta of Mobile Legends: Bang Bang, focusing on item interactions, hero roles, and build optimization strategies for both casual and competitive play.

TECHNICAL LIMITATIONS
  • No direct real-time interaction with Moonton's game servers.
  • Data dependency on periodic manual updates following game patches.
  • Statistical analysis is based on general meta trends, not individual player skill.
